# E-commerce Order Performance Dashboard

## Overview
Interactive Power BI dashboard developed to analyze e-commerce order performance, sales trends, customer behavior, and business KPIs.

## Tools Used
- Power BI
- Excel
- DAX
- Power Query

## Features
- KPI tracking dashboard
- Region-wise sales analysis
- Category performance analysis
- Profit and revenue insights
- Interactive filtering and visualization

## Insights Generated
- High-performing product categories
- Sales trends and customer patterns
- Business performance metrics
